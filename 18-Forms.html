HTML Input Form Attributes Examples

1. form Attribute

Allows an input to be part of a form even if it is placed outside the form.



2. formaction Attribute

Overrides the form's action URL for a specific submit button.





3. formenctype Attribute

Specifies how the form-data is encoded in POST requests.



4. formmethod Attribute

Defines whether form data will be sent using GET or POST for a specific button.





5. formtarget Attribute

Defines where to open the result after form submission.





6. formnovalidate Attribute

Submit form without validation for this button.



7. novalidate Attribute (Form Attribute)

Disables validation for the entire form.
